# راهنمای مشارکت در پروژه سینمای گروهی

سلام! خوشحال می‌شویم اگر شما هم در بهبود این پروژه کمک کنید. لطفاً قبل از ارسال هرگونه گزارش یا درخواست، این راهنما را مطالعه کنید.

## چگونه می‌توانید کمک کنید؟

- **گزارش باگ (Bug)**: اگر مشکلی در کار پروژه دیدید، از طریق [Issues](https://github.com/YOUR_USERNAME/watchparty/issues) گزارش دهید.
- **پیشنهاد قابلیت جدید**: ایده‌های خود را در بخش Issues مطرح کنید.
- **بهبود کد**: می‌توانید اشکالات را رفع کنید یا قابلیت جدید اضافه کنید و سپس Pull Request بفرستید.
- **بهبود مستندات**: خوانایی README، اضافه کردن توضیحات بیشتر یا رفع اشکالات تایپی.

## گزارش باگ (Bug Report)

هنگام گزارش باگ، لطفاً موارد زیر را ذکر کنید:
- مرحله به مرحله چطور مشکل را مشاهده می‌کنید.
- خروجی کنسول مرورگر (F12) اگر خطایی هست.
- خروجی ترمینال سرور.
- سیستمعامل و مرورگر خود.

## درخواست Pull Request (PR)

اگر می‌خواهید تغییری در کد ایجاد کنید:

1. ابتدا یک Issue باز کنید و توضیح دهید چه کاری می‌خواهید انجام دهید (مگر اینکه تغییر خیلی کوچک باشد).
2. مخزن را **Fork** کنید (در گیت‌هاب دکمه Fork).
3. در کامپیوتر خود، فورک شده را clone کنید:
   ```bash
   git clone https://github.com/YOUR_USERNAME/watchparty.git
   cd watchparty
