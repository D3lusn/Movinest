# 🎬 سینمای گروهی - WatchParty

پلتفرم تماشای هماهنگ فیلم و ویدیو به صورت گروهی با قابلیت چت، مدیریت اتاق، پنل ادمین و فرم نظرات.

## نسخه 9.3

### امکانات
- ساخت اتاق خصوصی با کد
- پخش هماهنگ ویدیو (YouTube، MP4، M3U8)
- چت گروهی
- پنل مدیریت اتاق (اخراج کاربر، همگام‌سازی)
- پنل ادمین اصلی (مدیریت اتاق‌ها و نظرات)
- ارسال اعلان سراسری
- فرم ثبت نظر و پیشنهاد

### نصب و اجرا
```bash
npm install
node server.js
```


# 🎬 Group Cinema - WatchParty

A platform for watching movies and videos in groups with chat capabilities, room management, admin panel and comment form.

## Version 9.3

### Features
- Create a private room with code
- Synchronized video playback (YouTube, MP4, M3U8)
- Group chat
- Room management panel (user removal, synchronization)
- Main admin panel (room and comment management)
- Send global notifications
- Comment and suggestion form

### Installation and execution
```bash
npm install
node server.js
